Build tools

Lean how you can harness the power of Bamburgh Z Admin Dashboard PRO to your advantage!

Setup

Bamburgh Z Admin Dashboard PRO uses NPM and Gulp to run tasks like starting a live relead web server, compiling SCSS files to CSS, minifying/concatenating Javascript files, automatically compressing image assets and many more.

To start using the included Gulp tasks you'll need to follow the steps below:

  1. 1. Download and install Node.js globally, which will be used to install all the necessary dependencies from the included package.json file.

  2. 2. After successfully completing the Node.js installation, navigate to folder where you downloaded the Bamburgh Z Admin Dashboard PRO source files and run the npm install command. Remember, the command needs to be ran in the root directly that also contains the package.json file.

After this, you'll be able to run various Gulp tasks to build your assets folder from the source code.

Gulp tasks

Run any of the following commands in a command line in the root of your installation. The root folder can be identified if it contains gulpfile.js file.

Task Description
gulp gulp starts a Browsersync instance on port 3000 served from the root folder, with the modifications you made in SASS files or HTML. It automatically reloads the pages when you modify any of the source code files.
gulp build gulp build creates the /assets directory with compiled files. The difference from gulp is that it doesn't start a livereload Browsersync web server.

Troubleshooting

If you encounter problems while installing dependencies, uninstall all previous dependencies versions, both global and local and run again npm install.
